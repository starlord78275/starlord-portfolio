Twitter Clone

The Twitter Clone project replicates the core features of the popular social media platform, Twitter. Users can post tweets, like and retweet posts, follow other users, and receive real-time updates. This app is built using the MERN stack (MongoDB, Express, React, Node.js).

Key Features

  • Real-time tweeting, liking, and retweeting
  • Follow and unfollow users
  • Real-time updates and notifications
  • User authentication with JWT tokens
