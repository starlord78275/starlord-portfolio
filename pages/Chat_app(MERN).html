Chat App (MERN)

The Chat App project is a real-time messaging platform built with the MERN stack (MongoDB, Express, React, Node.js). Users can create accounts, log in, and chat in real-time with other users. The app supports private and group chats, along with notifications for new messages. It also includes a user-friendly interface for sending and receiving messages, emojis, and attachments.

Key Features

  • Real-time messaging using WebSockets (Socket.IO)
  • Private and group chat functionality
  • User authentication with JWT tokens
  • Message notifications and attachments (e.g., images, files)
