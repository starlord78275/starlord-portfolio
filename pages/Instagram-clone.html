Instagram Clone

The Instagram Clone project recreates the core features of the popular social media platform, Instagram. Users can upload photos, follow other users, like posts, and leave comments. The app is built using the MERN stack (MongoDB, Express, React, Node.js) and includes real-time updates and notifications.

Key Features

  • Photo uploads with filters and captions
  • Follow/unfollow users and view their posts
  • Like and comment on posts
  • Real-time notifications and feed updates
